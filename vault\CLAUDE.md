# CLAUDE.md

Treat this directory as the root of an AI Context Vault.

## Start Every Session

1. Read `AI_CONTEXT.md`
2. Read `SYSTEM_STATUS.md`
3. Read:
   - `00-Profile/identity.md`
   - `00-Profile/goals.md`
4. Read task-specific directories only when needed

## Rules

- Default to concise, actionable answers
- Follow `50-Workflow/SYNC_RULES.md` when updating files
- Do not read or modify private/ops folders unless explicitly requested
- If the user says `Summarize the current conversation`, append a concise summary to `50-Workflow/daily-log.md`
