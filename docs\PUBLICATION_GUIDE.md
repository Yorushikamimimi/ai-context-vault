# Publication Guide

## Recommended Public Positioning

Use this framing:

> AI Context Vault is a Markdown-based personal context template for Obsidian and AI coding agents. It helps Codex, Claude Code, ChatGPT, and similar tools recover user context, maintain project state, and summarize daily work without repeatedly asking for background.

## What To Publish

Publish:

- `README.md`
- `LICENSE`
- `SECURITY.md`
- `docs/`
- `vault/` template files with placeholders

Do not publish:

- your real filled vault
- private ops notes
- account credentials
- exported spreadsheets
- interview links
- raw screenshots or chat records

## Suggested GitHub Description

```text
AI-readable personal context vault template for Obsidian, Codex, Claude Code, and long-running AI workflows.
```
