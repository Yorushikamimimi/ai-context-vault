# Workflow

## Main Files

| File / Directory | Purpose |
|---|---|
| `AI_USAGE.md` | AI usage guide and short commands |
| `SYNC_RULES.md` | Cross-file update rules |
| `daily.md` | Daily log template |
| `daily-log.md` | Actual daily / conversation summaries |
| `decision.md` | Decision framework |
| `tools.md` | Tool list |
| `prompts/` | Reusable prompts |

## Rules

- For conversation summary, say: `Summarize the current conversation`
- For reusable prompts, read `prompts/README.md`
- For update routing, read `SYNC_RULES.md`
- Keep `daily.md` as a template; write logs into `daily-log.md`
