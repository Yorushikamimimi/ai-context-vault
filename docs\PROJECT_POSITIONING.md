# Project Positioning

## One Sentence

AI Context Vault is a local-first Markdown template that lets AI tools understand your stable context, current priorities, project state, and workflow rules.

## Core Value

Most AI conversations start cold. This vault makes context explicit:

- Who you are
- What you are working on
- What the AI should read first
- Where updates should be written
- Which files are source of truth
- What should stay private

## Similar Tools

This template is adjacent to:

- Obsidian: local Markdown knowledge base
- Logseq: block-based personal knowledge management
- Notion: cloud workspace and databases
- Anytype: local-first object-based workspace
- Mem / Reflect / Reor: AI-assisted note systems

The difference is that AI Context Vault is not another note app. It is an AI-readable structure that can sit inside Obsidian or any Markdown folder.

## Best Use Cases

- Long-running coding agent sessions
- Career and interview preparation
- Project documentation and retrospectives
- Daily conversation summaries
- Cross-tool handoff between Codex, Claude Code, and ChatGPT
