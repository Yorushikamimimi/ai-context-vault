# Career

## Structure

- `00-Resume/`: resume drafts
- `10-Applications/`: application progress and target companies
- `20-Interviews/`: interview records, questions, prep notes
- `30-Job-Fairs/`: job fair plans and results
- `40-Growth/`: career growth retrospectives

## Source of Truth

- Applications: `10-Applications/application-progress.md`
- Interview records: `20-Interviews/records.md`
- Questions: `20-Interviews/questions.md`
- Targets: `10-Applications/target-companies.md`

## Rules

- Application changes go to `application-progress.md`
- Interview events go to `records.md`
- Preparation notes go to a dedicated prep file
- Keep execution files concise
