# Sync Rules

## General

| Scenario | Update |
|---|---|
| Identity changes | `AI_CONTEXT.md` and `00-Profile/identity.md` |
| Goals change | `00-Profile/goals.md` and maybe `SYSTEM_STATUS.md` |
| New important file or structure | `SYSTEM_STATUS.md` |
| Agent entry rule changes | `AI_CONTEXT.md` and `CLAUDE.md` |

## Career

| Scenario | Update |
|---|---|
| Application status changes | `30-Career/10-Applications/application-progress.md` |
| Interview scheduled / finished | `30-Career/20-Interviews/records.md` |
| Interview questions found | `30-Career/20-Interviews/questions.md` |
| Job fair plan | `30-Career/30-Job-Fairs/` |
| Resume changed | check `30-Career/00-Resume/resume-onepage.md` |

## Privacy

- Do not write secrets into public files.
- Put private data outside this public template or keep it in ignored directories.
